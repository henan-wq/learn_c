# 测试题

```
1
以下哪些是有效的变量名？

A.main

B.4ever

C.monkey-king

D.__int
正确答案：A、D你错选为A、C、D


2
给定：
int a,b;
scanf("%d %d", &a, &b);
则以下哪些输入方式是正确的？

A.1 2

B.1,2

C.1(回车)2

D.1、2
正确答案：A、C你选对了


3
给定：
int a,b; 
以下哪些scanf的使用是正确的？

A.scanf("%d", &a);
scanf("%d", &b);

B.scanf("%d %d", &a, &b);

C.scanf("%d, %d", &a, &b);

D.scanf("%d %d", a, b);
正确答案：A、B、C你错选为A、B


4
给定以下代码段：
int a,b=0;
则a的初始值是0

A.对

B.错误
正确答案：B你选对了


5
写出下式的运算结果：
10/3.0*3
10
正确答案：10.0


6
写出下式的运算结果：
10/3*3.0
9.0
正确答案：9.0
```
