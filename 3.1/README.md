# 3.1 判断

## if

满足条件才会执行大括弧内的内的语句：

```
if( 条件成立 ) {
  ...
}
```

## 判断的条件

条件

计算两个值之间的关系，所以叫做关系运算。

|  运算符  |  意义  |
|  ---  |  ---  |
|  = =  |  相等  |
|  !=  |  不相等  |
|  >  |  大于  |
|  >=  |  大于或等于  |
|  <  |  小于  |
|  <=  |  小于或等于  |

## 关系运算的结果

当两个值的关系符合关系运算符的预期时，
关系运算的结果为整数1，否则为整数0。

```
#include <stdio.h>

int main()
{
	printf("%d\n", 5==3);
	printf("%d\n", 5>3);
	printf("%d\n", 5<=3);

	return 0;
}
```

输出结果为：

```
0
1
0
```
