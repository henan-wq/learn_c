# 数的范围

- 对于一个字节（8位），可以表达的是：
    - 0000 0000 - 1111 1111
- 其中
    - 0000 0000 -> 0
    - 1111 1111 ~ 1000 0000 -> -1 ~ -128
    - 0000 0001 ~ 0111 1111 -> 1 ~ 127

```
#include <stdio.h>

int main()
{
    char c = 255;
    int i = 255;
    printf("c=%d, i=%d\n", c, i);

    return 0;
}
/*
输出结果如下：
c=-1, i=255
*/
```
